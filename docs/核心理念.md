# 拉麵屋 · 游戏核心理念

> 本文是项目的「设计北极星」。任何新功能在动手前先问一句:**它有没有让核心循环更紧、更有动力?** 如果没有,就先放一放。

---

## 一、一句话愿景

**在一条有烟火气的小街上,经营一家你自己的拉面店——从一口锅、一碗面做起,一步步把生意做大。**

类型定位:**2D 像素 · 开放小镇探索 + 拉面店经营**(经营/模拟为主轴,轻动作为调剂)。

---

## 二、核心幻想(玩家在追求什么)

玩家扮演一个**刚接手小店的年轻人**,核心幻想是:

- **「亲手做」的满足感**:不是点按钮,而是**提起汤、提起面、放进碗**——像真的在颠勺。
- **「把店做大」的成长感**:今天只有几个座位、几种配料;明天有更气派的店面、更复杂的招牌面、更多回头客。
- **「这条街是我的」的归属感**:从底商一间,到逐渐熟悉、点亮整条商业街。

> 一句话:**「治愈的烟火气」 + 「看得见的成长」**。

---

## 三、核心循环(The Core Loop)

游戏的一切都应服务于这个闭环:

```
        做拉面赚钱
           │
           ▼
        花钱变强 ── 升级店面 / 新配料 / 新设备
           │
           ▼
     做得更快 · 更高级 · 卖更贵
           │
           ▼
        赚更多 ──────┐
           ▲          │
           └──────────┘
```

- **赚(短循环,几十秒):** 一桌客人 → 看订单 → 提料组装 → 上菜 → 拿小费。
- **花(中循环,几分钟):** 攒够钱 → 回街上的商店 → 买升级。
- **强(长循环,整局):** 升级后效率/单价提升 → 解锁更难更赚的内容 → 朝总目标推进。

> 🔄 **方向调整(重要):** 本作**不做经营**,转向**「亲手做一碗面」的手作/解压游戏**。
> 上面的「赚→花→强」经营循环**已弃用**——移除了金钱压力、信誉清零、营业倒计时、升级商店。
> **新核心循环 = 手作循环:**
>
> ```
> 看客人要什么 → 一步步亲手做(煮面看火候 · 浇汤 · 码面 · 铺料)→ 上菜 → 「这碗做得好不好」的品相反馈 → 想把下一碗做得更好
> ```
>
> 已落地:**煮面火候/计时**(在绿色窗口提起,太生/太烂有反馈)+ 全面去压力(从容、无失败)。
> 下一步可深化手作:浇汤份量、码面、摆料位置/份量、更多面种汤底,把「做」的每一步都变成手感。

---

## 四、核心机制:提起 → 放入

这是本作**最有记忆点、不可替代**的设计,必须始终是体验的中心:

- 每个动作都是**两步**:**点一下提起**(汤勺/面/配料跟着光标)→ **点碗放入**。
- 底料固定(湯 + 麵 + 牛肉),客人只在乎 **要不要加 蔥花 / 香菜 / 辣椒**——规则极简,**上手 10 秒,压力来自速度**。
- 压力源:**耐心条**(客人会走)+ **营业倒计时**(一天)。

**设计准则:** 任何对做面玩法的扩展(新配料、新汤底、新工序),都要保留「**提起→放入**的手感」和「**规则一眼看懂**」这两条底线。不要把它变成复杂的配方记忆游戏。

---

## 五、经济与成长(让数字有意义)

- **金币是唯一货币**,来自上菜小费(越快、配料越对越多)。
- **金币必须有出口**——这是经营游戏的命脉。规划中的消费方向:
  - **店面升级**:加座位、加灶口、提升客人耐心、小费加成。
  - **配方/食材**:解锁新汤底、新配料、招牌面(更贵、更挑战)。
  - **门面/装潢**:点亮街道、吸引更高级客人。
- **总目标(给玩家一个跑的方向):** 例如「攒够 ¥X 盘下整条街」/「N 天内达成营业额」——有目标才有「为什么今天比昨天更努力」。

---

## 六、世界观与基调

- **场景:** 一条**小石板商业街**,拉面店是某栋大厦的**底商**;街坊邻里、坐满食客的店内,处处是**生活气**。
- **角色:** 一个**很酷的白发年轻人**(白T、利落造型),亲力亲为的小老板。
- **基调:** **温暖、轻松、治愈**为底色;偶尔来点**热闹与冒险**(如紫金大厦的小怪)调味,但不喧宾夺主。
- **美术:** 统一的**手作像素**风格,暖色、清爽、零碎细节(灯笼、暖帘、热气)。

---

## 七、支线的定位:为主线服务

游戏里所有「非做面」的内容(逛街、紫金大厦打怪等)都只是**调剂**,它们的存在价值取决于**能否回流到主循环**:

- **可以有冒险**(打小怪、探索),但它应当**掉金币 / 掉稀有食材**,让玩家「打完怪能做出更好的面、卖更高的价」。
- **反面教材:** 一段和赚钱、做面完全无关、打完没奖励的玩法——玩家没有理由碰它。

> 准则:**先闭合主循环,再加支线;支线必须有一根线接回主线。**

---

## 八、设计取舍(Do / Don't)

**要(Do)**
- 守住「提起→放入」的手感与「规则一眼懂」。
- 每加一个系统,先想清楚它在「赚 / 花 / 强」哪一环。
- 保持治愈、轻松的整体节奏。
- 让玩家的努力**可见**:钱变成了看得见的升级/装潢。

**不要(Don't)**
- 不要堆叠互不相干的小玩法(广而不深)。
- 不要让金币变成「只涨不用」的空数字。
- 不要把简单爽快的做面,做成繁琐的记忆/配方负担。
- 不要让支线难度/复杂度盖过主线。

---

## 九、一句话总结

> **核心是「亲手做拉面把小店做大」。**
> 守住「提起→放入」的手感,接上「赚钱→花钱→变强」的闭环,让这条小街因为玩家的努力而一点点热闹起来——其余一切,都为这件事服务。
