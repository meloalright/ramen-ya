# 拉麵屋 · 游戏核心理念(v2 · 美式卡通画风)

> 本文是项目的「设计北极星」。任何新功能/新素材动手前先问:**它符合新画风、并让核心循环更有动力吗?**
>
> **本次重写的核心变化:美术方向从「16-bit 像素风」→「美式卡通画风」。玩法内核保留(亲手做拉面 + 自由小镇),换的是「皮」和「气质」。**

---

## 〇、画风转向(本版重点)

**弃用**:像素 / 16-bit / 瓦片降采样。
**改为**:**美式卡通 · 西方手绘插画风**。

### 这套风格长什么样(给 AI / 美术的硬定义)
- **粗而干净的黑色描边**(bold clean outlines)包裹每个物体。
- **明快、饱和**的色彩;大色块为主 + **柔和 cel-shading**(分明的高光/阴影块,不是写实渐变)。
- **夸张、有弹性的造型**:大头、大手、圆润外形、Q 而不腻;道具也圆鼓鼓有"重量感"。
- **手绘质感**:轻微笔触和材质噪点,但**不要照片写实、不要 3D 渲染感**。
- **表情和动作夸张、有喜感**:客人馋到流口水、上菜"哐"地一声满足、煮过头冒黑烟。
- **暖色主调**,保留拉面店的烟火气,但用卡通的活力与幽默重新表达。
- 参考坐标(给个范围,不照抄):现代西方动画(Cartoon Network 系)、Supercell/西方手游卡通、《Don't Starve》的粗线条手绘(但更暖更圆)、《Overcooked》的明快卡通厨房。

### 技术影响
- **不再降采样到 16×16 瓦片**;改用**高分辨率插画**直接绘制/贴图,允许更高视口分辨率。
- 角色走路、客人、UI 都走卡通插画(干净描边 + cel 阴影)。
- AI 提示词的【统一风格】前缀已同步改为美式卡通(见 `素材AI提示词.md`)。
- 字体可从像素中文字体换成**更圆润的卡通字体**(可选)。

---

## 一、一句话愿景

**在一条热闹的小馆街上,亲手做出一碗碗让人馋到不行的拉面——用美式卡通的活力和幽默,把"下厨的爽快"放大。**

类型定位:**2D 美式卡通 · 自由小镇探索 + 亲手做面(手作/解压为主轴,轻探索调剂)。**

---

## 二、核心幻想(玩家在追求什么)——基本不变

- **「亲手做」的爽快**:不是点按钮,而是**煮面看火候 · 浇汤 · 码面 · 撒料**——像真的在颠勺,且每一步反馈夸张、解压。
- **「越做越好」的成长**:从手忙脚乱到行云流水;品相越来越好看。
- **「这条街有我一份」的归属**:逛街、串门、点亮小馆街。

> 一句话:**「热乎乎的烟火气」 + 「夸张爽快的手感」 + 「美式卡通的幽默」。**

---

## 三、核心循环(The Core Loop)

```
看客人要什么(夸张表情) → 一步步亲手做(煮面火候 · 浇汤 · 码面 · 撒料)
   → 上菜(夸张满足反馈) → 「这碗做得好不好」的品相反馈 → 想把下一碗做得更好、更快、更帅
```

- 本作**不做重经营**(无金钱压力 / 信誉清零 / 营业倒计时):核心是**手作 + 解压 + 幽默**。
- 已落地的手作:**煮面火候**(绿窗口提起)、**浇汤**(一勺满)、**撒料**(握罐在碗上拖着撒)、**俯视微倾料理台**、两口**大缸** + 料盘。
- 下一步深化(对标厨房卡通游戏):切牛肉/叉烧、煮蛋计时、**品相星级 ★**、客人夸张反应。

---

## 四、核心机制:提起 → 放入(保留)

- 每个动作两步:**点一下提起**(料跟着光标)→ **点目标放入/操作**。
- 底料固定(湯 + 麵 + 牛肉),客人只在乎 **要不要加 蔥花 / 香菜 / 辣椒**——规则一眼懂。
- 准则:任何新工序都要守住「**提起→放入**的手感」「**规则一眼看懂**」,并且**表现要卡通、要有反馈喜感**。

---

## 五、基调与世界观(气质升级)

- **从「治愈像素烟火气」→「温暖 + 幽默 + 活力 的美式卡通日常」。**
- **角色**:一个有点酷又有点呆萌的小老板(夸张比例、表情丰富);客人各具夸张性格。
- **场景**:热闹的小馆街,拉面馆是其中一间;街坊、坐满的食客、冒着热气的大锅——都用卡通的夸张和暖意呈现。
- **冒险调剂**(如紫金大廈打小怪)可保留,但同样卡通化、轻松搞笑,不喧宾夺主。

---

## 六、资产迁移(从像素 → 美式卡通)

> 现有所有美术(主角、建筑、地块、做面物件、店内家具、档口背景、图标)都需**按新画风整体重做**。流程不变:AI 生成 → 我切片/抠底/缩放/导入。

需要重出的清单(沿用 `素材AI提示词.md`,但换成美式卡通【统一风格】):
- 主角行走精灵、客人/食客、商业街建筑、树/道具、做面物件(碗/缸/料盘/手持)、地块、店内家具、档口背景、App 图标、UI 元素。

---

## 七、设计取舍(Do / Don't)

**要(Do)**
- 守住「提起→放入」手感与「规则一眼懂」。
- **一切表现都卡通化、夸张化、有反馈喜感**(动作、表情、音效)。
- 保持暖色、热闹、解压的整体节奏。

**不要(Don't)**
- 不要回到像素或写实/3D 渲染感。
- 不要把简单爽快的做面做成繁琐配方负担。
- 不要堆叠互不相干的小玩法。
- 不要让支线盖过"做面"主线。

---

## 八、一句话总结

> **核心是「用美式卡通的活力,把『亲手做一碗好拉面』的爽快放到最大」。**
> 玩法内核不变(亲手做 + 自由小镇 + 解压),变的是**画风(像素→美式卡通)和气质(治愈→热闹幽默)**——其余一切,为这件事服务。
